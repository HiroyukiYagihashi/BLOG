### TailwindCSS
使うのが初めてだったのですがスムーズに使用できた&好感触でした。
CSSライブラリ特有の癖が出なくて良いと思った。
今後の個人開発のデザインはTailwindCSSを主体に使っていきたい。
styled-componentsとの組み合わせは後日やります。([これ](https://github.com/ben-rogerson/twin.macro) でできるっぽい)

ちなみに、AMPで使用する際はひと工夫が必要ですのでご注意を。(使ってるCSSだけ圧縮してstyleタグ内で展開する)

### AMP
Accelerated Mobile Pagesの略で、対応するとモバイルぺージの表示がめちゃくちゃ爆速になる。
AMP CacheとかいうAMPページを配信するためのCDNによってキャッシュされる。
リリース当初は記事ページのみをサポートする規格だったが現在は様々なサイトが導入している。
ちなみにPCでの表示も爆速になる。

### PWA
Progressive Web Apps。
WebサイトやWebアプリをネイティブアプリのように使えるようになる技術。
モバイルのイメージが強いですがデスクトップアプリでの導入例も最近はかなり多い印象があります。
プッシュ通知やアイコン追加、Webでネックだったオフラインでの動作も安定させることができます。
(速くiOSのプッシュ通知対応してくれ)
